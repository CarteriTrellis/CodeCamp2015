Seattle Code Camp 2015

September 12th, 2015 at Seattle University's Pigott Building

Their Exits and Their Entrances: Actors and the .NET Framework

What are Actors and how can they help you build highly-scalable, reliable web services? We will start with a brief overview of the history and theory behind Actors, then compare a couple of different implementations of Actors on .NET. We will end the session with a working C# example using Akka.NET. This session will assume some familiarity with web development concepts and the .NET family of languages (C# in particular), but should still be accessible to novices.

Thanks for checking out the materials for my Seattle Code Camp 2015 presentation. In addition to my slides and as-prepared remarks, you should find credits and references for all images and source materials used in the session, the source code used in the session, and some cheat sheets on setting pre-requisites for the demo software and cheat sheets for the code itself that I used during the session.
